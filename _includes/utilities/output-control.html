Output control

Control whether or not to output any USWDS utility family by setting the value of the $[utility_family]-settings.output variable in your USWDS settings configuration. This code sample shows a how to exclude the display utilities from your CSS:

@use "uswds-core" with (
  $display-settings: (
    output: false,
  ),
);

A value of true will output the utility family and any variants. A value of false prevents the output of the family and any variants.
